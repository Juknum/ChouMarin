- This project has been developped under Eclipse & Visual Studio Code
- Compile project using make:  

  ```bash
  cd ./ChouMarin/Debug; make all
  ```

- Run the project:  

  ```bash
  ./ChouMarin/Debug/ChouMarin.exe
  ```
